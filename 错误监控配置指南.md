# 错误监控配置指南

## 方案一: 使用 Sentry (推荐)

### 1. 注册 Sentry 账号
访问: https://sentry.io/signup/

### 2. 创建项目
- 登录后点击 "Create Project"
- 选择平台: "Browser JavaScript"
- 输入项目名称: "libra-blog"
- 点击 "Create Project"

### 3. 获取 DSN
创建项目后,会看到类似这样的代码:
```javascript
Sentry.init({
  dsn: "https://xxxxxx@xxxxx.ingest.sentry.io/xxxxxx",
});
```

复制这个 DSN 地址

### 4. 配置到项目
打开 `app.js`,找到这一行:
```javascript
// TODO: 注册 Sentry 后,将你的 DSN 填在这里
// this.errorMonitor.initSentry('YOUR_SENTRY_DSN_HERE');
```

改成:
```javascript
this.errorMonitor.initSentry('你的DSN地址');
```

### 5. 查看错误
- 访问 https://sentry.io
- 登录后进入你的项目
- 在 "Issues" 页面可以看到所有错误
- 每个错误都包含:
  - 错误消息和堆栈
  - 用户浏览器信息
  - 发生时间
  - 用户操作记录

### 免费额度
- 每月 5,000 个错误事件
- 1 个项目
- 7 天数据保留
- 对个人博客完全够用

## 方案二: 使用 Supabase 记录错误 (备用)

如果不想用 Sentry,可以把错误记录到你的 Supabase 数据库。

### 1. 创建错误日志表
在 Supabase 控制台执行:

```sql
CREATE TABLE error_logs (
  id BIGSERIAL PRIMARY KEY,
  error_message TEXT,
  error_stack TEXT,
  context JSONB,
  user_agent TEXT,
  url TEXT,
  timestamp TIMESTAMPTZ DEFAULT NOW()
);

-- 添加索引
CREATE INDEX idx_error_logs_timestamp ON error_logs(timestamp DESC);
```

### 2. 修改 ErrorMonitorService.js
在 `logToSupabase` 方法中,取消注释数据库插入代码:

```javascript
async logToSupabase(error, context = {}) {
    if (!this.supabaseService) return;
    
    try {
        const errorLog = {
            error_message: error.message || String(error),
            error_stack: error.stack || '',
            context: context,
            user_agent: navigator.userAgent,
            url: window.location.href
        };
        
        await this.supabaseService.supabase
            .from('error_logs')
            .insert([errorLog]);
            
    } catch(e) {
        console.error('Failed to log error to Supabase:', e);
    }
}
```

### 3. 查看错误
在 Supabase 控制台的 Table Editor 中查看 `error_logs` 表

## 测试错误监控

可以在浏览器控制台运行:
```javascript
throw new Error('测试错误监控');
```

然后去 Sentry 或 Supabase 查看是否记录了这个错误。

---

## 注意事项

1. Sentry 的 DSN 是公开的,可以直接写在代码里
2. 不要在错误日志中记录敏感信息(密码、token等)
3. 定期检查错误日志,及时修复问题
4. 免费版 Sentry 有流量限制,超过后会停止记录
